What I learned:

During my NCL competitions, I built proficiency across several investigative and exploitation categories:
- Network Forensics - Reassebmbling PCAPs, protocol decoding (DNS,SMB,HTTP), XOR extraction, tunnel detection
- Cryptography & Data Recovery - AES-CBC key identification, brute-force ops, hash cracking(NTLM,RACF,bcrypt)
- Log Analysis & SIEM-style Thinking - Pattern spotting, alert prioritization, event correlation
- Web Exploitation - File inclusion, CSRF, auth bypass, XXS, python pickling, SQL Injection 
- OSINT & Recon - Metadata mining, API enumeration, subdomain mapping

These skills directly reinforced my GCIA, GPEN, and GWAPT-level training, and helped me apply tools like Wireshark, Burp Suite, Autopsy, Hashcat, Python scipting, John The Ripper, Nmap, Dig, SQLmap, and more in real-world contexts
